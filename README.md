# 🚗 RideNow – Uber Inspired Frontend

A modern, responsive ride-booking landing page inspired by Uber’s design.  
Built purely with HTML and CSS to practice layout structuring, responsive design, and UI styling.

> ⚠️ This project is created for educational purposes only and is not affiliated with or endorsed by Uber.

---

## 🌐 Live Demo

🔗 https://jahanvi1405.github.io/uber-clone/

---

## ✨ Features

- Clean and modern landing page UI
- Responsive design (works on desktop & mobile)
- Navigation bar with styled buttons
- Hero section with call-to-action
- Styled ride booking interface
- Organized CSS structure

---

## 🛠 Tech Stack

- **HTML5**
- **CSS3**
- Git & GitHub for version control
- GitHub Pages (for deployment)
